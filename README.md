flappybird
==========

FlappyBird clone with LimeJS

###To rebuild :
  - Extract folder next to your limejs folder.
  - Install Grunt.
  - Install python (2.7)
  - Using "grunt build" in cmd or terminal
-----
###To run demo:
  - Extract folder next to your limejs folder.	
  - Install Grunt.
  - Install python (2.7)
  - Using "grunt server" in cmd or terminal
